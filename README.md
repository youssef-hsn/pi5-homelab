## Philosophy

This repository is more than just a collection of configs and scripts. It's a **living record** of my homelab evolution, built on three core principles:

1. **Resource Efficiency First** - Every service, every config, every decision is made with resource consumption in mind. Raspberry Pi OS Lite provides the lean foundation, and every addition is carefully considered.

2. **Open Source by Default** - This is my reference, but it's also yours. If my setup changes significantly (different hardware, different approach), I'll fork this repo and maintain both versions open source. Knowledge should be shared, not hoarded.

3. **Learn by Doing** - This is a playground. Things will break, get rebuilt, and evolve. The documentation reflects real-world usage, not theoretical perfection.

## Hardware

- **Board**: Raspberry Pi 5 (8GB RAM)
- **Cooling**: Passive cooling fan only
- **OS**: Raspberry Pi OS Lite (64-bit)

*Note: If my hardware setup changes, I'll fork this repository to maintain separate documentation for different configurations.*
